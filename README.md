# SIMPLE-WEATHER-APP
🌦️ Weather App

A simple and responsive weather application that allows users to check the current temperature of any location in real time.

🚀 Features
🌍 Search weather by city name
🌡️ Displays current temperature in Celsius
⚡ Fast and real-time data using Weather API
❌ Error handling for invalid locations
🎨 Clean and modern UI
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
Weather API (https://www.weatherapi.com/
)
📸 Demo

📂 Project Structure
📁 Weather-App
│── index.html     # Main HTML file
│── style.css      # Styling
│── script.js      # App logic
⚙️ How It Works
User enters a location
App sends a request to the Weather API
Fetches real-time weather data
Displays the current temperature
🔑 API Usage

This app uses the WeatherAPI service:

http://api.weatherapi.com/v1/current.json?key=YOUR_API_KEY&q=LOCATION&aqi=yes

⚠️ Note: Replace YOUR_API_KEY with your own API key.

▶️ How to Run
Clone the repository:
git clone https://github.com/your-username/weather-app.git
Open the project folder
Run index.html in your browser
⚠️ Important Note
The API key is currently exposed in script.js.
For production, store it securely (e.g., backend or environment variables).
